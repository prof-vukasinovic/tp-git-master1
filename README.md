# TP Git/GitHub - Master 1 Informatique

Ce dépôt est utilisé pour le TP de prise en main de Git/GitHub.

## Consignes générales
- Chaque étudiant doit créer un fichier `nom_etudiant.txt` dans le dossier de l'année en cours.
- Utilisez des branches pour isoler vos modifications.

## Structure du dépôt
- `template/` : Fichiers de base pour le TP.
- `2025-2026/` : Dossier pour l'année en cours.
